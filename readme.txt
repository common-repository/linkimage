A plugin for displaying images from media gallery with custom links.

== Description ==
This plugin gives you all the controls for every attached image and give the link to it. 
Comes with custom image styling for padding and margin to create a better image position.

<h3><strong>Key Features & options</strong></h3>
<ul>
	<li>Dragable images</li>
	<li>Ajax powered panel</li>
	<li>Easy custom style for each image</li>
	<li>Powerful widget options</li>
	<li>Advanced image styles</li>
	<li>Intro & outro text to add additional text or HTML.</li>
	<li>Custom styles and scripts to add additional CSS style or javascript per widget ID.</li>
</ul>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>Go to the menu 'Plugins' -> 'Install' and search for 'Linkimage'</li>
<li>Click 'install'</li>
</ul>
